# GreenFlow-CO2-Calculator
CO2 Calculator for SMEs => Small and Medium Enterprises (SMEs)
Automated CO2 Calculation: Calculation of Scope 1 (Diesel, Natural Gas) and Scope 2 (Electricity) emissions based on integrated emission factors.
Dynamic Branding: Automated generation of a color-coordinated logo derived from the company name. => Optional Extensions (Time Permitting):
Visual Analytics: Integration of charts within the PDF report to visualize the distribution of emission sources.
Professional Reporting: Generation of compliant PDF documents featuring automated layout, pagination, and a system-generated footer/disclaimer.
